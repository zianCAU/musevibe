# 随身乐器「缪斯 MUSE」使用指南

硬件：Otto-Robot ESP32-S3 摄像头版

---

## 一、硬件接线

| 按钮 | 和弦 | 接在哪里 |
|---|---|---|
| BTN1 | **C 大三** | 舵机插针 left_hand 的 Signal 针（GPIO 4） |
| BTN2 | **Am 小三** | 舵机插针 left_leg 的 Signal 针（GPIO 5） |
| BTN3 | **F 大三** | 舵机插针 left_foot 的 Signal 针（GPIO 6） |
| BTN4 | **G 大三** | 舵机插针 right_hand 的 Signal 针（GPIO 7） |

每个触摸按钮只需把导线焊到 Signal 针，另一端接铜箔片（直径 10–15mm 即可）。ESP32-S3 内部自带上拉，无需额外电阻。

---

## 二、主界面

开机后进入主界面：

```
┌──────────┬──────────────────────────────┐
│          │  AI 对话消息                  │
│  🎵      │                              │
│  MUSE    │  [ C ] [ Am ] [ F ] [ G ]   │
└──────────┴──────────────────────────────┘
```

- 左侧：缪斯 AI 头像（紫色光晕呼吸动效）
- 右上：和 AI 的对话记录
- 右下：4 个和弦按钮，触按时高亮

---

## 三、触摸弹奏

**直接用手指触碰对应铜箔片即可发声。**

- 轻触 → 松开时播放和弦（钢琴分解音色）
- 持续按住 1 秒以上，只有 **BTN1（C）** 和 **BTN4（G）** 有特殊功能（见下方翻页）

---

## 四、和 AI 对话

唤醒设备后，可以说：

### 日常聊天

| 说什么 | 缪斯会做什么 |
|---|---|
| 「来弹一段」「给我演示个和弦进行」 | 自动依次播放 C → Am → F → G |
| 「换成扫弦」 | 改为扫弦音色（全部和弦叠加播放） |
| 「换回分解弦」 | 改回默认分解弦音色 |
| 「升一个调」/ 「降两个调」 | 移调，范围 ±3 半音 |
| 「回到主界面」 | 从歌词界面返回主界面 |

### 景色填词（主要功能）

1. 把摄像头对着窗外或任意景色
2. 说：**「帮我基于这个景色写首歌」** 或 **「拍一张，给我填词」**
3. 缪斯会：
   - 拍摄当前画面，描述景色
   - 以景色为灵感，生成歌词+和弦谱
   - 自动切换到歌词界面

也可以指定参考曲目：
> 「以这个景色为灵感，参考《同桌的你》的旋律填词」

## 五、歌词弹唱界面

AI 生成歌词后自动切换：

```
┌─────────────────────────────────────────┐
│ ♪ 基于《同桌的你》·景色填词    2 / 4段  │
├─────────────────────────────────────────┤
│ [C ] 那片山坡住着谁的故事   ← 暗淡已过  │
│ [Am] 秋风把叶子卷成信纸     ← 暗淡已过  │
│ ▌[F ] 你说过窗外的光是温柔的 ← 当前行   │
│ [G ] 我还记得那一天的云               │
│ [Am] 也许青春就是一场风景             │
├─────────────────────────────────────────┤
│    C     Am   [F]    G                 │  ← 底部高亮当前和弦
└─────────────────────────────────────────┘
```

- 当前行：紫色背景高亮 + 左侧竖线标记
- 已唱过的行：半透明暗淡
- 底部和弦条：实时高亮当前行用的和弦

**翻页操作：**

| 操作 | 效果 |
|---|---|
| 长按 BTN1（C）≥ 1 秒 | 向上翻页 |
| 长按 BTN4（G）≥ 1 秒 | 向下翻页 |
| 说「回到主界面」 | 退出歌词界面 |

**弹唱方法：** 看屏幕左边的和弦标签，按照提示按对应按钮，跟着节奏唱。

## 六、语音指令速查

| 指令 | 效果 |
|---|---|
| 「来弹一段」 | AI 演示 C-Am-F-G 进行 |
| 「换成扫弦」 | 切换为扫弦音色 |
| 「换回分解弦」 | 切换为分解弦音色 |
| 「升一个调」 | 整体升 1 个半音 |
| 「降两个调」 | 整体降 2 个半音 |
| 「恢复原调」 | 回到 0 移调 |
| 「帮我写首歌」 | 拍照 + AI 景色填词 |
| 「回到主界面」 | 退出歌词界面 |

---

## 七、注意事项

- 触摸灵敏度：导线不超过 20cm，铜箔直径建议 10–15mm
- 摄像头拍照需要联网（用于 AI 视觉描述），请确保设备已连接 Wi-Fi
- 歌词生成约需 5–15 秒，期间屏幕显示「正在创作...」
- 目前支持和弦：**C、Am、F、G**（覆盖 90%+ 流行歌曲）
